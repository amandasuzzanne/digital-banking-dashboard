# Digital Banking Dashboard (Frontend)

A responsive digital banking dashboard built with React and Vite, simulating core banking user interfaces such as account summaries, transaction history, authentication flows, and secure navigation.
The project focuses on clean UI architecture, performance, security awareness, and real-world frontend best practices.

ive Demo: https://digital-banking-dashboard-nj7u8x535-amandasuzzannes-projects.vercel.app/

Tech Stack: React · JavaScript (ES6+) · Vite · CSS · REST APIs

## Getting Started (Local Development)
1️⃣ Clone the repository  
git clone https://github.com/amandasuzzanne/digital-banking-dashboard.git  
cd digital-banking-dashboard

2️⃣ Install dependencies  
npm install

3️⃣ Run the app  
npm run dev

The app will be available at:  
http://localhost:5173
